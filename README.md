## Apa Ada Di Sini

| Folder | Isi |
|--------|-----|
| `instruction/` | Custom instruction untuk Claude.ai Kids Project |
| `artifacts/` | Permainan dan alat interaktif (HTML) |
| `worksheets/` | Lembaran kerja dan kuiz yang boleh dicetak |

## Cara Guna (Claude.ai Project)

1. Buka [claude.ai](https://claude.ai) → buka project **ARH Kids**
2. Pergi ke **Edit project details → Custom Instructions**
3. Salin teks dari `instruction/claude-kids-system-prompt.md`
4. Simpan

## Cara Buka Artifak

Muat turun fail `.html` dari folder `artifacts/` → buka dengan mana-mana pelayar web (Chrome, Edge).
Tidak perlu internet selepas dimuat turun.
